Introduction

Cycle Analytics has been created to provide both real-time and historical analysis of Barclays cycle stations (aka Borris bikes) within London.

In total there are around 700 stations within London where these bicycles can be rented from, each station updates a government server every 30 seconds with information reguarding information about the station such as; bicycles availabile or empty slots available (to drop cycles off).

To view the source files for this project, as well as contribute, please see the official GitHub project page here GitHub project page
